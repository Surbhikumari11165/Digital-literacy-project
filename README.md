##  Project Overview
This project was completed as part of the Digital Literacy course. As a Student Digital Ambassador, the objective was to create a portfolio that demonstrates awareness of digital tools, safe online practices, professional communication, and cyber safety.

The project includes five tasks covering infographic creation, digital portfolio setup, coding platforms, email etiquette, and cybercrime awareness.
